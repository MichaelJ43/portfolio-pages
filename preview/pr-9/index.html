How this website is built

Technical aspects demonstrated by the portfolio site itself.

Static generation

A small Crystal generator reads YAML content, renders ECR templates, and publishes static HTML that can run on GitHub Pages or a custom domain.

Shared design assets

The page consumes versioned tokens, shell styles, and primitives from static.michaelj43.dev so multiple projects can share one visual system.

Auth-aware shell

The top bar is provided by a shared script that talks to the auth platform for sign-in state while keeping this portfolio static.

Analytics client

A lightweight analytics script identifies this app and sends page events to the shared API platform.

Routing and delivery

Public demos, preview builds, and portfolio pages use custom domains, static hosting, and GitHub Actions-based deployment.

Skills

A YAML-driven skill index that can grow with the portfolio.
